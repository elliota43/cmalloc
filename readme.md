# cmalloc

Exploring various memory allocation strategies as a means to familiarize myself
with lower level systems programming.

Some of the resources I am using:

* Ginger Bill's [Memory Allocation Strategies](https://www.gingerbill.org/series/memory-allocation-strategies) (shoutout Ginger Bill, I love [Odin](https://odin-lang.org)).
* This dude's [blog](https://mohitmishra786.github.io/exploring-os/src/) I randomly stumbled across.
* Another [random website](https://memory-pool-system.readthedocs.io/en/latest/index.html) that has a metric ton of documentation on memory pools.
* [Memory Management Reference](https://www.memorymanagement.org/mmref/begin.html)
* [Everything About Memory Allocators](https://mohitmishra786.github.io/chessman/2024/11/24/Everything-About-Memory-Allocators-Write-a-simple-memory-allocator.html)
* [samwho](https://samwho.dev/memory-allocation/) who deserves a shoutout for putting work into visually representing memory on his blog!
* and, of course, [OSDev](https://wiki.osdev.org/Memory_Allocation)
